# 《不存在的住户》全流程与完美通关

> 完整剧透。本文既是走查清单，也是修改进度门槛时的开发参考。

## 流程总览

```text
开场记忆与醒来
  → CJ-0713登录
  → 1204空置房与失联儿童
  → 1304事故、顾小满与顾长河残留会话
  → 1104周明川遗体与注销账号
  → 恒目流程证据
  → 1404投诉与主角身份
  → 记忆覆盖反制
  → 下班：提交全部证据 / 重新进入循环
```

游戏故意要求玩家搜索怀疑的关键词，而不是按固定目录顺序阅读。下文给出的是保证不漏证据的完美路线，不代表UI应把它显示成任务清单。

调查台账按房间归并为四个章节。对应事实推导完成前，台账和推导目录只显示房间号；完成后才揭示章节名：`空房间与隐形孩子`、`没离开的人`、`周明川，最后一次呼叫`、`明天，再一次`。

房号是入口词，不是全案目录。直接搜索时，索引应保持以下范围：

| 房号 | 房号检索的最大范围 |
| --- | --- |
| 1204 | 夜间投诉、空置巡检、顶面渗漏排查 |
| 1304 | 顶面渗漏排查、顾长河表层回访、关联的1204投诉；许芷遥获救后新增墙面修补前影像 |
| 1104 | 1304章节完成前不显示内部材料；读取周明川同步摘要后，房号检索新增房间实况、同步摘要与墙体联合记录 |
| 1404 | 前期最多出现一条破碎关怀索引；最终章节只增加投诉工单，不列出后续身份档案 |

服务排班、公开通报、儿童身份、事故附件、异地证明、四份1404加密材料等都必须用上游正文给出的二级词找到。

## 0. 开场与登录

1. 观看三段日常记忆：厨房晚饭、雨中等车、周末整理衣物。
2. 文案从日常过渡到生命脆弱和意识错位，随后从梦境醒来。
3. 点击进入身份认证终端。
4. 使用工牌进入，或选择密码登录：`CJ-0713` / `0713`。
5. 登录瞬间短暂飘红闪烁，然后进入正常的 `物业管理系统`。

验证点：

- 开场不是营销首页，第一屏就是演出。
- 开场、普通登录与结局使用忧伤阴郁的D小调主题；进入物业系统后切换为冷静、低动态、带稀疏半音摩擦和反向钟声感的环境循环。暗红403警告、周明川账号登录、本地档案和身份侦测阶段改用带急促双脉冲、噪声升压与金属撞击的惊悚主题。音乐在首次用户操作后开始，静音偏好跨刷新保留。
- 播放监控视频或四轨声纹样本时，背景音乐自动压低，不能遮蔽案件原始音频。
- 新证据实际写入调查台账时播放一次独立的低音确认提示；重复核验、刷新和恢复旧存档不得重复播放。
- 刷新在hash路由和本地存档允许的范围内恢复当前页面。
- 登录页的 `遗忘` 二次确认后清除全部本机进度并回到初始状态。
- 未恢复周明川凭据之前，账号选择中不能直接使用 `ZM-0602`。

## 1. 1204：从投诉到儿童救援

### 1.1 打开起始工单

打开 `1204 夜间滴水投诉复核`，记住：

- 每天 `00:04` 开始，`00:10` 结束，共六分钟。
- 1304近24小时用水为零。
- 报修人怀疑楼上有人，但系统不能直接替玩家核验其身份。

首页的“待办工单”始终只显示一项，并在玩家完成当前档案的附件核验、表单、路线编排或真相推导后刷新为下一份应处理档案。它不是完整任务清单，也不会提前列出后续事实。除系统直接派发的1204起始工单和最终1404投诉外，首次点击待办档案只定位到对应检索结果，仍由玩家从结果中打开记录；已读档案可从待办卡直接返回。留言回复、真相推导和注销账号登录等非档案节点会临时占用同一张待办卡。

可以先查看留言板。林若岚、许先生、0702和1302的留言混合了线索与噪声。留言按发信人归入会话组，同一人的内容按剧情时间轴从早到晚排列；包含最新一条消息的会话组排在最前。

### 1.3 受理失联事件并复核监控异常

1. 打开失联儿童紧急留言，分别询问“最后在哪里见到她？”和“报警回执是什么？”。

3. 接警事件受理后，安防中心才生成独立的公共区域录像保全任务。搜索 `监控`、`录像保全`、`DL-0713-0041` 或 `00:04`，打开 `12层公共区域事件回放复核`。
4. 播放串联视频；播放失败时仍可用逐帧复核。
5. 只多选异常时间节点：`00:04`、`00:07`、`00:10`、`00:12`。
6. 提交后取得 `wetFootprints`。

四个节点的含义：

- `00:04`：1204门磁和潮湿双足痕迹。
- `00:07`：消防门/13层网关形成路径。
- `00:10`：回放丢帧或恢复节点。
- `00:12`：录像缓存出现重复序列；不是画面里出现了人。

只有执行“跨表核验 / 服务授权与门禁”并取得 `vacancyMismatch`，失联儿童紧急留言才触发。监控复核发生在失联事件之后，不能反过来作为事件的触发条件；单纯阅读文章仍不会触发。
