# 1304 field recording sources

The four interactive stems use real recordings obtained from Freesound. Each source page marks the recording as Creative Commons 0 (CC0 1.0). The committed MP3 files are derived from Freesound's public previews. The humming stem is trimmed to 18 seconds, shifted slightly upward while preserving tempo, filtered, and given a short bathroom reflection; it remains a real human humming performance with no generated voice or replacement melody.
